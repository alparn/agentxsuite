Getting Started with AgentxSuite

Learn how to set up and configure your first MCP agent orchestration with AgentxSuite. This comprehensive guide walks you through installation, configuration, and your first agent deployment.

Introduction

AgentxSuite provides a powerful platform for orchestrating Model Context Protocol (MCP) agents at enterprise scale. This guide will walk you through the essential steps to get started with your first agent deployment.

Prerequisites

  • Python 3.11 or higher
  • PostgreSQL 14+ (SQLite works for development)
  • Redis (for rate limiting and caching)
  • Basic understanding of MCP protocol

Installation

Clone the repository and install dependencies:

git clone https://github.com/alparn/agentxsuite
cd AgentxSuite/backend
pip install -r requirements/base.txt

Configuration

Set up your environment variables:

export SECRET_KEY="your-secret-key"
export FERNET_KEY="your-fernet-key"
export DATABASE_URL="postgresql://user:pass@localhost/agentxsuite"

Your First Agent

Create your first organization, environment, and agent through the Django admin or API. Configure MCP connections and define security policies before executing your first run.
